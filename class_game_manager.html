Descripción detallada

Esta clase controla el estado del juego.

Funciona como helper para todas las clases, ya que controla el UI, las camaras, la muerte del jugador, el guardado y la carga de savegames, entre otros.

Se debe instanciar antes de usar GameManager.Instance
